Building meshes with rollply

Alexandre GÃ©nin

28/10/2014

Rollply uses internally a grid of coordinates (a mesh). For each points of this mesh it selects the observations falling within the window, then applies the function using this subset. The user can either provide a mesh as a data.frame or rollply will take care of building one automatically.

Several helper functions are provided to build nice 2D meshes, they all start with build_mesh_.

  • build_mesh_grid_identical: builds a grid of regularly-spaced points, with the same number of points on each dimension.

  • build_mesh_grid_proportional (2D only): build a 2D grid of points, with a number of points on each dimension proportional to the total span of that dimension in the observations.

  • build_mesh_ahull_crop (2D only): builds a 2D grid of points, then discard all the points that do not fall in the alphahull of the actual data available.

  • build_mesh_ahull_fill (2D only): same as above, except iterating to build a mesh with a final number of points approximately equal to the one asked for (parameter mesh.res in function)

# We request a mesh with approximately this number of points
npts <- 500
base.plot <- ggplot(NULL, aes(x,y)) + 
               geom_point(data=mdw, shape='+') +
               xlab('UTM X') + 
               ylab('UTM Y')

meshes <- list(grid_identical    = build_mesh_grid_identical(mdw[ ,c('x','y')], npts),
               grid_proportional = build_mesh_grid_proportional(mdw[ ,c('x','y')], npts),
               ahull_crop        = build_mesh_ahull_crop(mdw[ ,c('x','y')], npts),
               ahull_fill        = build_mesh_ahull_fill(mdw[ ,c('x','y')], npts))
  • build_mesh_grid_identical:
nrow(meshes[['grid_identical']])
## [1] 484
base.plot + geom_point(data=meshes[['grid_identical']])

  • build_mesh_grid_proportional:
nrow(meshes[['grid_proportional']])
## [1] 500
base.plot + geom_point(data=meshes[['grid_proportional']])

  • build_mesh_ahull_crop:
nrow(meshes[['ahull_crop']])
## [1] 184
base.plot + geom_point(data=meshes[['ahull_crop']])

  • build_mesh_ahull_fill: takes much longer to run but gives a number of points that correspond approximately to what was requested
nrow(meshes[['ahull_fill']])
## [1] 482
base.plot + geom_point(data=meshes[['ahull_fill']])
